SUMA

Sumar es juntar 2 o más cosas para saber cuántas hay en total.









RESTA

Restar es quitar una cantidad a otra que ya teníamos. Lo contario a la suma









MULTIPLICACIÓN

La multiplicación es una forma muy rápida de sumar el mismo número una y otra vez. Por ejemplo, 2 x 3, podemos sumar el número 2, tres veces ( 2 + 2 + 2) o podemos hacerlo de forma mágica y veloz conociendo las tablas de multiplicar.









DIVISIÓN

Una división es una operación matemática que consiste en dividir un número en partes iguales. Veamos un ejemplo: 50 : 5 (cincuenta entre cinco) es repartir 50 en 5 partes iguales. Una división es la operación inversa a la multiplicación.

ÁREA CIRCULO

El área de un círculo se refiere a la medida de la superficie delimitada por una circunferencia.









ÁREA CUADRADO

El área del cuadrado es igual a lado por lado.









ÁREA TRIÁNGULO

El área de un triángulo es igual a base por altura partido por 2. La altura es la recta perpendicular trazada desde un vértice al lado opuesto (o su prolongación).









ÁREA RECTÁNGULO

El área del rectángulo es igual a base por altura..
